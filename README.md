# PROJETO-PVI

Projeto acadêmico desenvolvido para a disciplina **Projeto de Vida Integrador (PVI) – 2º Semestre**.

## Visão Geral
Desenvolvimento de uma **landing page profissional para fotógrafa**, focada em divulgação digital, portfólio visual e apresentação de serviços.  
O projeto prioriza identidade visual, navegação fluida, responsividade e usabilidade.

## Objetivo
Criar um site estático funcional que reúna:
- Portfólio fotográfico organizado por categorias
- Informações profissionais e serviços
- Interface clara, moderna e responsiva
- Boa experiência do usuário em diferentes tamanhos de tela

## Tecnologias Utilizadas
- **HTML5** – estrutura das páginas  
- **CSS3** – layout, responsividade e identidade visual  
- **JavaScript** – navegação, interações e validações  

## Metodologia
- Prototipagem inicial no **Figma** (layout, cores e estrutura)
- Conversão do protótipo para HTML e CSS
- Implementação de scripts em JavaScript
- Testes contínuos no navegador para garantir:
- Responsividade
- Coerência visual
- Usabilidade
- Desenvolvimento colaborativo com revisões frequentes

## Funcionalidades
- Landing page principal
- Galeria de imagens dividida por categorias
- Navegação entre páginas
- Scripts para interação do usuário
- Estrutura organizada por pastas

## Estrutura do Projeto
- `landing page-main/` – páginas HTML principais  
- `style/` – arquivos CSS  
- `script/` – arquivos JavaScript  
- `fotos do site/` – imagens e galerias  

## Resultados
O projeto resultou em uma landing page:
- Funcional e visualmente consistente
- Adequada para uso profissional
- Alinhada à identidade da fotógrafa
